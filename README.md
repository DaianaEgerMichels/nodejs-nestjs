# Rest API - NestJS

## Description

This repository contains a REST API developed with the NestJS framework, providing a modular, scalable, TypeScript-based architecture. The API was created to meet the specific needs of the project.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
